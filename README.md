# Caja Registradora en Python

Proyecto desarrollado para aplicar funciones, diccionarios y lógica de control de flujo.  
Simula el funcionamiento de una caja real con gestión de usuarios y múltiples métodos de pago.

## Funcionalidades
- **Gestión de Stock:** Diccionario con precios y descuentos.
- **Cálculo:** IVA (21%) y totales automáticos.
- **Sistema VIP:** Descuento del 25% con login.
- **Pagos:** Efectivo (vuelto) y Tarjeta (10% recargo).

## Ejecución
```bash
python caja_registradora.py
